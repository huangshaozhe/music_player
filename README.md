## 安装使用：
        1.运行Solve-slow-download.bat。
        2.运行Module-installer.py,安装需要的模块。      
        3.运行main.py，使用音乐播放器。
        4.可在shell命令提示符上输入:pyinstaller -F -w main.py，生成EXE可执行文件。
